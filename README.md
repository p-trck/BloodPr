# 🩺 혈압 관리 웹앱

`혈압관리.html` 파일 하나로 동작하는 혈압 기록 앱입니다. 설치 없이 브라우저에서 바로 사용합니다.

## 사용 방법

`혈압관리.html`을 더블클릭해서 열면 됩니다. 데이터는 그 브라우저에 자동 저장됩니다.

**스마트폰에서 쓰려면**: 파일을 카카오톡·이메일 등으로 폰에 보내 열거나, 구글 드라이브에 올려 열 수 있습니다. 브라우저 메뉴에서 "홈 화면에 추가"를 하면 앱처럼 사용됩니다.

> 데이터는 브라우저별로 따로 저장됩니다. 여러 기기에서 쓰려면 아래 구글 시트 연동으로 백업·병합하세요.

## 주요 기능

- **휠 피커 입력**: 숫자를 타이핑하지 않고 휠을 굴려서 입력. 마지막 입력값에서 시작하므로 몇 칸만 움직이면 됩니다.
- **혈압 단계 색상**: 대한고혈압학회 기준으로 정상/주의혈압/고혈압전단계/1기/2기를 색으로 표시.
- **기간별 그래프**: 7일/30일/90일/전체, 아침·저녁 필터, 120/80·140/90 기준선 표시.
- **통계**: 전체·아침·저녁 평균, 최고/최저, 단계 분포, 복약률.
- **복약 체크 + 메모**: 약 복용 여부와 컨디션 메모 기록.
- **구글 시트 백업**, **CSV 내보내기/가져오기**.

## 구글 시트 연동 설정 (약 5분)

1. [sheets.google.com](https://sheets.google.com)에서 **새 스프레드시트**를 만듭니다. (이름 예: "혈압기록")
2. 메뉴에서 **확장 프로그램 → Apps Script**를 클릭합니다.
3. 편집기에 있는 기존 코드를 모두 지우고, `구글시트-연동코드.gs` 파일의 내용을 **전체 복사해 붙여넣고** 저장(💾)합니다.
4. 오른쪽 위 **배포 → 새 배포**를 클릭합니다.
   - 유형 선택(⚙️) → **웹 앱**
   - 액세스 권한: **모든 사용자** ← 중요!
   - **배포** 클릭
5. 권한 승인 창이 나오면: 계정 선택 → "확인되지 않은 앱" 경고에서 **고급 → 이동** → **허용**. (본인이 만든 스크립트라 안전합니다)
6. 발급된 **웹 앱 URL**(`https://script.google.com/macros/s/.../exec`)을 복사합니다.
7. 혈압 앱의 **⚙️ 설정**에 URL을 붙여넣고, 원하면 **자동 백업**을 켭니다.

이후 "지금 시트로 내보내기"를 누르면 시트에 전체 기록이 저장되고, 다른 기기에서는 "시트에서 가져오기"로 병합할 수 있습니다.

> 코드를 수정한 뒤에는 **배포 → 배포 관리 → 새 버전**으로 다시 배포해야 반영됩니다.

## 혈압 단계 기준 (대한고혈압학회)

| 단계 | 수축기 (mmHg) | | 이완기 (mmHg) |
|---|---|---|---|
| 저혈압 | < 90 | 또는 | < 60 |
| 정상 | < 120 | 그리고 | < 80 |
| 주의혈압 | 120–129 | 그리고 | < 80 |
| 고혈압전단계 | 130–139 | 또는 | 80–89 |
| 고혈압 1기 | 140–159 | 또는 | 90–99 |
| 고혈압 2기 | ≥ 160 | 또는 | ≥ 100 |
