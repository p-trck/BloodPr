# 🩺 혈압 관리 웹앱

`혈압관리.html` 파일 하나로 동작하는 혈압 기록 앱입니다. 설치 없이 브라우저에서 바로 사용합니다.

## 사용 방법

`혈압관리.html`을 더블클릭해서 열면 됩니다. 데이터는 그 브라우저에 자동 저장됩니다.

**스마트폰에서 쓰려면**: 파일을 카카오톡·이메일 등으로 폰에 보내 열거나, 구글 드라이브에 올려 열 수 있습니다. 브라우저 메뉴에서 "홈 화면에 추가"를 하면 앱처럼 사용됩니다.

> 데이터는 브라우저별로 따로 저장됩니다. 여러 기기에서 쓰려면 아래 구글 시트 연동으로 백업·병합하세요.

## 주요 기능

- **휠 피커 입력**: 숫자를 타이핑하지 않고 휠을 굴려서 입력. 마지막 입력값에서 시작하므로 몇 칸만 움직이면 됩니다.
- **혈압 단계 색상**: 대한고혈압학회 기준으로 정상/주의혈압/고혈압전단계/1기/2기를 색으로 표시.
- **기간별 그래프**: 7일/30일/90일/전체, 아침·저녁 필터, 120/80·140/90 기준선 표시.
- **통계**: 전체·아침·저녁 평균, 최고/최저, 단계 분포, 복약률.
- **복약 체크 + 메모**: 약 복용 여부와 컨디션 메모 기록.
- **구글 시트 백업**, **CSV 내보내기/가져오기**.

## 구글 시트 연동 설정

앱에서 **Google 계정 연결** 버튼만 누르면 "혈압기록" 시트가 자동 생성되고 동기화됩니다.
다만 최초 1회, 구글에 "이 앱을 허용한다"는 등록(Client ID 발급)이 필요합니다. 한 번만 하면 모든 기기에서 로그인 버튼 하나로 사용합니다.

### 1단계. Client ID 발급 (최초 1회, 약 10분)

1. [console.cloud.google.com](https://console.cloud.google.com) 접속 → 구글 계정 로그인
2. 상단 프로젝트 선택 → **새 프로젝트** → 이름 "혈압앱" → 만들기
3. 왼쪽 메뉴 **API 및 서비스 → 라이브러리**에서 아래 2개를 검색해 각각 **사용 설정**:
   - Google Sheets API
   - Google Drive API
4. **API 및 서비스 → OAuth 동의 화면** (또는 "Google Auth Platform"):
   - User Type: **외부(External)** → 만들기
   - 앱 이름 "혈압관리", 본인 이메일 입력 → 저장
   - **게시 상태**를 **프로덕션(게시)** 으로 변경 ← 테스트 상태로 두면 7일마다 재로그인해야 합니다
5. **API 및 서비스 → 사용자 인증 정보 → 사용자 인증 정보 만들기 → OAuth 클라이언트 ID**:
   - 애플리케이션 유형: **웹 애플리케이션**
   - 승인된 자바스크립트 원본에 추가: `https://p-trck.github.io`
   - 만들기 → 표시되는 **클라이언트 ID**(`xxxx.apps.googleusercontent.com`) 복사

### 2단계. 앱에서 연결

1. `https://p-trck.github.io/BloodPr/` 접속 → **⚙️ 설정**
2. Client ID 붙여넣기 → **🔗 Google 계정 연결** 클릭
3. 구글 로그인 → 권한 허용 → 끝! "혈압기록" 시트가 자동으로 만들어집니다.

이후 "시트로 내보내기 / 시트에서 가져오기" 버튼을 쓰거나 **자동 백업**을 켜두면 됩니다. 다른 기기에서는 같은 Client ID 입력 후 로그인만 하면 같은 시트를 공유합니다.

> 팁: `index.html` 상단의 `DEFAULT_CLIENT_ID = ''` 에 발급받은 ID를 넣어 배포하면, 어떤 기기에서도 ID 입력 없이 로그인 버튼만 누르면 됩니다.

**주의**: 구글 로그인은 웹 주소(https)로 접속했을 때만 동작합니다. `index.html` 파일을 더블클릭해 열면(file://) 시트 연동은 사용할 수 없고 나머지 기능만 동작합니다.

## 혈압 단계 기준 (대한고혈압학회)

| 단계 | 수축기 (mmHg) | | 이완기 (mmHg) |
|---|---|---|---|
| 저혈압 | < 90 | 또는 | < 60 |
| 정상 | < 120 | 그리고 | < 80 |
| 주의혈압 | 120–129 | 그리고 | < 80 |
| 고혈압전단계 | 130–139 | 또는 | 80–89 |
| 고혈압 1기 | 140–159 | 또는 | 90–99 |
| 고혈압 2기 | ≥ 160 | 또는 | ≥ 100 |
